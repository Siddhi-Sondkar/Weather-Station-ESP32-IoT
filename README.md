# 🌦️ Weather Station using ESP32

## 📌 Overview
An IoT-based Weather Monitoring System designed to monitor environmental conditions and provide real-time alerts using Telegram.

---

## 🚀 Features
- 🌡 Real-time weather monitoring  
- 💨 Wind speed & direction detection  
- 🌧 Rainfall measurement  
- 🚨 Threshold-based alerts  
- 📲 Telegram notifications  
- 💾 Data logging using SD card  
- 📍 Location tracking (GPS API)  

---

## 🛠️ Components Used
- ESP32  
- Wind Speed Sensor  
- Rain Sensor  
- Wind Direction Sensor  
- LCD Display (I2C)  
- SD Card Module  

---

## ⚙️ Working
- Sensors continuously collect environmental data  
- Data is processed by ESP32  
- Alerts are triggered when thresholds are exceeded  
- Weather updates are sent every 5 minutes via Telegram  
- Data is logged into SD card
